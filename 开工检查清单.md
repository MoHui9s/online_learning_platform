## ⚠️ 开工前必须完成的环境准备

### PM 需要做的（启动会前或同步进行）

```bash
# 1. 启动 Docker 基础设施（必须有 Docker Desktop）
docker compose up -d mysql redis chromadb
# 验证：docker compose ps（三个容器都是 Up 状态）

# 2. 初始化数据库
cd backend
conda run -n edu_platform alembic upgrade head
# 验证：alembic current 显示 6a1c0f34e2a6

# 3. 启动后端
conda run -n edu_platform uvicorn app.main:app --reload --port 8000 &
# 验证：curl http://localhost:8000/health

# 4. 启动前端
cd frontend
npm install  # 如果有新人加入
npm run dev &
# 验证：curl http://localhost:5173
```

如果 Docker Desktop 未安装或无法启动，最低替代方案：
- MySQL 可以用 SQLite 临时替代（修改 `app/core/database.py` 的 DATABASE_URL）
- Redis 测试环境已 Mock（conftest.py 实现了 mock redis）
- ChromaDB 仅 BE-C 需要，可以等上午再启动
